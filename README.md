## requirements

* Python >=3.7

## Installing

```bash
git clone https://github.com/gravures/gravure.git
cd gravure && pip install . 
```

## Building documentation

```bash
pip install myst_parser sphinxcontrib-apidoc
setup.py build_sphinx
```
